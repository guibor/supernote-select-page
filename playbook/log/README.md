# Playbook Log

Use this folder for durable decisions, research, and changelog entries.

## Naming convention

Use:

```text
purpose_topic_timestamp.md
```

Where:

- `purpose` is one of `decision`, `research`, or `changelog`
- `topic` is a short slug
- `timestamp` is `YYYY-MM-DD` or `YYYY-MM-DD-HHmm`

## Examples

- `decision_auth-provider_2026-04-05.md`
- `research_pricing-page_2026-04-05.md`
- `changelog_v0.3.0_2026-04-05.md`
